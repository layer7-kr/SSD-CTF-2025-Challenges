# 차이의 합

## 문제

$N$개의 정수로 이루어진 배열 $A$가 주어졌을 때 $A$의 원소들을 재배열하여 얻을 수 있는 인접한 원소의 차의 합의 최댓값을 구하시오.

## 입력

첫번째 줄에 $N$이 주어진다. $N$은 1보다 크고 200000보다 작거나 같은 자연수이다.

두번째 줄에 배열 $A$의 $i$번째 원소 $A_i$가 차례대로 공백으로 구분되어 주어진다. $A_i$는  2000000보다 작거나 같은 자연수이다.

## 출력

차이의 합의 최댓값을 출력하시오.

## 예제

### 입력 1

```
1
1 2 3
```

### 출력 1

```
3
```
